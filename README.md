# 易懂的雷达信号处理教程

> An Easy-to-Understand Radar Signal Processing Tutorial for Students and Engineers

## 📖 项目简介

这是一本面向学生和工程师的雷达信号处理入门教材。本书用通俗易懂的语言，从零开始讲解雷达的基本原理、信号处理方法和实际应用。

**特点：**
- ✅ 通俗易懂，避免过度数学化
- ✅ 从实际问题出发，循序渐进
- ✅ 配有丰富的图示和案例
- ✅ 适合自学和教学使用

## 📚 目录

### 第一章：雷达是什么
- 电磁波的"看"和"听"
- 距离和速度的测量原理
- 为什么需要信号处理

### 更多章节持续更新中...

## 🎯 适合人群

- 电子工程、通信工程专业的学生
- 从事雷达相关工作的工程师
- 对雷达信号处理感兴趣的自学者
- 需要快速了解雷达基础的研究人员

## 🚀 如何使用

1. 按章节顺序阅读，每章都建立在前一章的基础上
2. 重点理解物理意义，而不是死记公式
3. 结合图示理解抽象概念
4. 遇到不懂的地方可以先跳过，后面会有更详细的解释
