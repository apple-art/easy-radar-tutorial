## 📚 Tutorial Content

This tutorial is designed to help beginners quickly master the fundamentals of radar signal processing. Starting from basic concepts, it gradually introduces core principles and practical implementation techniques, enabling readers to build a solid understanding of radar systems through theory combined with MATLAB examples.

The tutorial currently covers essential topics in radar signal processing, including signal models, matched filtering, detection algorithms, and Doppler processing. Each chapter includes detailed explanations, illustrated diagrams, mathematical derivations, and corresponding MATLAB code to help readers grasp the concepts through hands-on practice.

**Main topics include:**

1. Radar basics and signal models
2. Matched filtering and pulse compression
3. CFAR detection and threshold determination
4. Doppler processing and velocity measurement

More advanced topics such as synthetic aperture radar (SAR) imaging, array signal processing, and tracking algorithms will be added in future updates. Contributions and suggestions are welcome.

**Tutorial features:**

1. Beginner-friendly with step-by-step progression from easy to advanced
2. Theory and practice combined with rich examples
3. Open-source MATLAB code for experimentation

## ✨ Features

This tutorial emphasizes intuitive understanding and practical application. Complex mathematical derivations are explained with clear physical interpretations to help readers build an intuitive grasp of the concepts.

All code examples are implemented in MATLAB and can be run directly. Readers can deepen their understanding by modifying parameters and observing results, making it an ideal resource for self-study and teaching.

The tutorial is published as open educational content under a Creative Commons license (with some restrictions—see the License section). Both HTML and PDF formats are provided for convenient reading across devices.

## 📂 Project Structure

- `index.html` / `chapters/` – Main tutorial content (HTML format, with Chinese pages maintained as bilingual counterpart; English is the default web entry)
- `pdf/` – PDF version of the tutorial
- `matlab/` – Complete MATLAB example code
- `matlab/basic_radar_equation.m` – Radar equation calculation
- `matlab/pulse_compression.m` – Pulse compression simulation
- `matlab/fmcw_range_processing.m` – FMCW radar ranging
- `matlab/cfar_detection.m` – CFAR detection algorithm
- `matlab/doppler_processing.m` – Doppler processing and velocity estimation
