# 易懂的雷达信号处理教程

> An Easy-to-Understand Radar Signal Processing Tutorial for Students and Engineers

## 项目简介

这是一本面向学生和工程师的雷达信号处理入门教材。本书尝试用通俗易懂的语言，从零开始讲解雷达的基本原理、信号处理方法和实际应用。

**特点：**
- 通俗易懂，避免过度数学化
- 从实际问题出发，循序渐进
- 配有图示、案例和 MATLAB 示例
- 适合自学和教学参考

## 仓库内容

- `matlab/第二章`：信号、采样相关练习
- `matlab/第三章`：雷达基础练习
- `matlab/第四章`：脉冲压缩示例
- `matlab/第五章`：多普勒与 FMCW 示例
- `matlab/第七章`：波束与测角示例
- `matlab/第八章`：检测与处理链示例
